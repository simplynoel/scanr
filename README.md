# Scanr

A lightweight QR code scanner and generator built with React Native, Expo, and SQLite.

Scan QR codes instantly, generate custom QR codes for various types of content, and keep a local history of your activity—all stored directly on your device.

## Features

- Scan QR codes using the device camera
- Generate QR codes from text and links
- Save scan history locally with SQLite
- View and reuse previously generated QR codes
- Export and share QR codes
- Save QR code images to the device
- Clean and responsive mobile interface

## Built With

- React Native
- Expo
- SQLite
- Expo Camera
- Expo File System
- Expo Image Picker
- Expo Media Library
- React Native QR Code SVG

## Installation

Clone the repository:

```bash
git clone https://github.com/simplynoel/scanr.git
```

Install dependencies:

```bash
npm install
```

Start the project:

```bash
npx expo start
```

## Project Goals

Scanr was created as a simple utility app focused on speed, usability, and offline functionality. Instead of relying on external services, scan history and generated QR codes are stored locally using SQLite.
